# Product Overview

## 1. Purpose
Define product vision, target audience, core features for the MVP of Group Free Time Scheduler.

## 2. Product Vision
Build a modern, simple, and mobile-first web app to help groups of friends find common free time effortlessly. The app focuses on real-time availability matching with a minimal UI inspired by Linear, Notion, and Discord.

## 3. Core Features (MVP)
### Authentication
- Google Login only (Supabase Auth).
- Auto-create user profiles upon first login.

### Group System
- Create a group.
- Join a group using an invite code.
- Leave a group.

### Availability Management
- Users can select free dates.
- Users can pick specific free time slots.
- Real-time updates.

### Matching & Realtime
- Display times where all/most members are free.
- Highlight best matching slots.
- Instant UI updates when any member modifies their availability.

## 4. UI/UX Guidelines
- **Mobile-first**: Bottom navigation on mobile, responsive layouts.
- **Aesthetic**: Minimal, Modern, Simple, Dark mode support.
- **Components**: Calendar selection UI, Time slot picker, Group availability overview, Realtime indicator.
